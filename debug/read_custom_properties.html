Office.js Custom Properties Reader

This tool reads custom properties from the current email using Office.js API.

How to use this tool:

  1. Open an email in Outlook
  2. Open this HTML file in Outlook's taskpane or a local web server
  3. Click the "Read Custom Properties" button below
  4. The tool will display the email's custom properties (summary, actions, etc.)

Note: This tool uses Office.js API and must run within Outlook's context.
